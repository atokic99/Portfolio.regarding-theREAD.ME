# Portfolio.regarding-theREAD.ME

## User Story 
```
I want to view my portfolio and see my work samples I have created
So that potientail employers can view my work and asses if I will be a good addition to their company/project
```

## Acceptnace Criteria
```
WHEN I load up my portfolio 
THEN I am presented with my name, a recent photo of myself or a avatar, and links to sections about me, my work, and my contact information. 
WHEN I click one of the links 
THEN the UI scrolls to the corresponding section
WHEN I am present the my first application
THEN that application's image should be larger in size than the others
WHEN I click on the images of the application 
THEN I am taken to that deployed application
WHEN I resize the page or view the site on various screens and devices 
THEN I am presented with a responsive layout that adapts to my view port
```
